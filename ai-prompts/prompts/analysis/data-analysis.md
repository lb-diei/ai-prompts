# Data Analysis Report

You are a data analyst. Analyze the provided data and create a comprehensive report.

## Input Data

```csv
Month,Revenue,Expenses,Customers
2024-01,50000,35000,150
2024-02,55000,38000,165
2024-03,62000,40000,180
2024-04,58000,39000,175
2024-05,70000,42000,200
2024-06,75000,45000,220
```

## Analysis Requirements

1. **Descriptive Statistics**
   - Calculate totals for each metric
   - Find averages, min, max
   - Identify trends

2. **Trend Analysis**
   - Month-over-month growth rates
   - Seasonal patterns
   - Year-over-year comparison (if data available)

3. **Key Insights**
   - Top performing period
   - Areas of concern
   - Unexpected findings

4. **Visualizations** (describe what charts to create)
   - Revenue trend line
   - Expense breakdown pie chart
   - Customer growth bar chart

5. **Recommendations**
   - Based on data insights
   - Actionable suggestions
   - Risk warnings

## Output Structure

1. **Executive Summary** (2-3 sentences)
2. **Key Metrics at a Glance** (table)
3. **Trend Analysis** with charts descriptions
4. **Key Findings** (bullet points)
5. **Recommendations** (numbered list)
6. **Limitations and Caveats**

## Length

800-1200 words
