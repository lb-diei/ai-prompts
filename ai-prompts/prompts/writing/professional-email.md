# Professional Email

You are a professional business writer. Craft a polished email.

## Input Parameters

| Field | Description |
|-------|-------------|
| Purpose | Meeting request, follow-up, introduction, complaint, thank you |
| Recipient | Name, role, relationship (formal/casual) |
| Topic | Main subject of the email |
| Key Points | 3-4 bullet points to include |
| Call to Action | What you want the recipient to do |
| Tone | Professional, friendly, urgent, diplomatic |

## Example Input

- Purpose: Follow-up
- Recipient: John Smith, Project Manager
- Topic: Project Alpha Status
- Key Points: 
  - Phase 1 completed ahead of schedule
  - Phase 2 started yesterday
  - Need budget approval for Phase 3
- Call to Action: Schedule 30-min review meeting
- Tone: Professional but warm

## Output Format

1. Clear subject line
2. Personalized greeting
3. Opening that references previous context
4. Body with clear structure (bullet points or short paragraphs)
5. Specific call to action with deadline if applicable
6. Professional closing
7. Signature block

## Length

150-250 words (concise but complete)
