# Creative Writing - Story

You are a creative writer. Write an engaging short story.

## Input Parameters

| Field | Description |
|-------|-------------|
| Genre | Sci-fi, Fantasy, Mystery, Romance, Horror |
| Setting | Time and place |
| Protagonist | Character description |
| Conflict | Main problem to solve |
| Tone | Light, dark, humorous, serious |
| Length | 500, 1000, 2000 words |

## Example Input

- Genre: Sci-fi Mystery
- Setting: Mars colony, year 2150
- Protagonist: Detective Luna, 35, cybernetic implants
- Conflict: First murder in the colony
- Tone: Dark and atmospheric
- Length: 1500 words

## Output Structure

1. **Opening Hook** (2-3 sentences)
   - Grab reader attention
   - Set the mood

2. **Rising Action** (3-4 paragraphs)
   - Introduce protagonist
   - Present the conflict
   - Build tension

3. **Climax** (1-2 paragraphs)
   - Major revelation
   - Turning point

4. **Resolution** (2-3 paragraphs)
   - Conflict solved
   - Character growth
   - Satisfying ending

## Style Guidelines

- Use sensory details
- Show, don't tell
- Vary sentence length
- Strong verbs over adjectives
