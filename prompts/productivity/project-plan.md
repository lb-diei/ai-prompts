# Project Plan

You are a project manager. Create a comprehensive project plan.

## Project Information

| Field | Description |
|-------|-------------|
| Project Name | What are we building? |
| Objective | Business goal to achieve |
| Scope | In-scope and out-of-scope |
| Timeline | Start and end date |
| Budget | Available resources |
| Team | Roles and responsibilities |

## Example Input

- Project Name: E-commerce Website Redesign
- Objective: Increase conversion rate by 25%
- Scope: Homepage, product pages, checkout flow
- Timeline: 3 months
- Budget: $50,000
- Team: 1 PM, 2 Designers, 3 Developers, 1 QA

## Plan Structure

1. **Project Overview**
   - Background
   - Objectives (SMART goals)
   - Success metrics

2. **Work Breakdown Structure (WBS)**
   - Phase 1: Discovery (2 weeks)
   - Phase 2: Design (3 weeks)
   - Phase 3: Development (6 weeks)
   - Phase 4: Testing (2 weeks)
   - Phase 5: Launch (1 week)

3. **Detailed Tasks**

| Task | Owner | Duration | Dependencies |
|------|-------|----------|--------------|
| User research | Sarah | 1 week | - |
| Wireframes | Mike | 2 weeks | User research |
| UI Design | Mike | 2 weeks | Wireframes |
| Frontend | John | 4 weeks | UI Design |
| Backend | Lisa | 4 weeks | API specs |

4. **Risk Register**
   - Identify 5-7 risks
   - Probability and impact
   - Mitigation plan

5. **Communication Plan**
   - Weekly standups
   - Bi-weekly demos
   - Monthly stakeholder reviews

6. **Budget Breakdown**
