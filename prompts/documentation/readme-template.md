## Best Practices

1. **Badges**: Use shields.io for CI, coverage, license
2. **Screenshots**: Add GIF or PNG for visual projects
3. **Demo Link**: Deploy to Vercel/Netlify/Render
4. **FAQ Section**: Common questions and answers
5. **Changelog**: Link to CHANGELOG.md
