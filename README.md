# AI Prompts Collection

A collection of high-quality prompts for Claude Code, ChatGPT, and other AI assistants. Boost your productivity with pre-crafted prompts.

## What It Does

| Before | After |
|--------|--------|
| Generic questions | Optimized AI responses |
| Time-consuming trial and error | Instant high-quality output |
| One-off conversations | Reusable prompt templates |

## Categories

### Coding Prompts
- Debug and fix code
- Write tests
- Code review
- Refactor for performance
- Explain complex algorithms

### Writing Prompts  
- Professional emails
- Technical documentation
- Creative writing
- Translation and localization

### Analysis Prompts
- Data analysis
- Market research
- Trend identification
- Risk assessment

### Productivity Prompts
- Meeting summaries
- Project planning
- Task prioritization
- Time management

## Quick Start

```bash
# Clone the repository
git clone https://github.com/lb-diei/ai-prompts.git

# Browse prompts by category
ls prompts/

# Use a prompt
cat prompts/coding/debug-fix.md
```

## Usage

1. Find your category
2. Copy the prompt
3. Paste to your AI assistant
4. Get optimized results!
